Para videos com explicação de varios projetos, o link do youtube pode se referir a apenas um intervalo do video
EX: https://www.youtube.com/embed/chElHV99xak?start=53&end=59

GET
api/filtro/tbGrupos/Pai/2
api/filtro/1/2
retorna os grupos que possuem o pai com id = 2

GET
api/filtro/tbProjetos/Pai/3
api/filtro/2/3
retorna os projetos que possuem o pai com id = 2

GET
api/filtro/tbProjetos/Tag/Madeira
api/filtro/3/3
retorna os projetos que possuem a tag Madeira
